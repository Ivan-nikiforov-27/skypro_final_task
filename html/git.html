Git

  • git status — посмотреть статус файлов, какие из них изменили, удалили или добавили в проект.
  • git checkout - позволяет переключиться с одной ветки на другую.
  • git info - С помощью этой команды можно получить подробную информацию о репозитории.
  • git init - используется для создания нового локального репозитория.
  • git branch - позволяет создавать, просматривать, переименовывать и удалять ветки.
